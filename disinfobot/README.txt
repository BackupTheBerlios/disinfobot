       Disinfobot
     ==============
     
Disinfobot runs from the source directory.  Simply copy "infobot.xml.example"
to "infobot.xml", and run "./infobot" (Unix) or "infobot" (Windows) to start it.
Make sure that the Java JVM program 'java' is in your path.
